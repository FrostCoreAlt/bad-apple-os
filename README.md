# bad-apple-os
Operating system strictly designed to play the Bad Apple video in 80x25 text mode.
# Compiling
Type ```make```, there should be generated an ISO-9660 ```badapple.iso``` file with the GNU GRUB bootloader
To run under QEMU, type ``make run``, an additional debug run can be executed with ```make debug``` where you can attach your debugger remotely.
